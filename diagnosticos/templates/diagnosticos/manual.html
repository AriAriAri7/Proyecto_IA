Introducción

El Sistema Experto para Diagnóstico de Fallas Críticas en Computadores es una herramienta inteligente diseñada para ayudar al personal administrativo de la Unidad Educativa Estatal "Prado del Norte II" a identificar y solucionar problemas comunes en los equipos de cómputo.

El sistema utiliza un motor de inferencia basado en reglas de producción que analiza los síntomas reportados y genera un diagnóstico preciso con recomendaciones de solución.

Nota: No se requieren conocimientos técnicos para usar el sistema. Solo debe responder las preguntas que aparecen en pantalla.

Requisitos del Sistema

  • Navegador web moderno (Chrome, Firefox, Edge, Safari)
  • Conexión a Internet (solo para carga inicial de librerías)
  • Resolución de pantalla mínima: 320px (móvil)
  • JavaScript habilitado en el navegador

¿Cómo usar el sistema?

1 Acceda al sistema

Abra su navegador web y diríjase a la dirección donde está instalado el sistema.

2 Inicie un diagnóstico

Haga clic en el botón "Iniciar Diagnóstico" en la página principal o en el menú de navegación.

3 Seleccione el síntoma principal

Elija de la lista el síntoma principal que presenta el equipo. Por ejemplo: "No enciende", "Pantalla azul", "Equipo lento", etc.

4 Responda las preguntas

El sistema le hará una serie de preguntas en lenguaje sencillo. Seleccione la opción que mejor describa lo que observa en el equipo.

5 Obtenga el diagnóstico

Una vez respondidas todas las preguntas, el sistema mostrará el diagnóstico con la falla identificada y las recomendaciones para solucionarla.

Síntomas y Categorías

El sistema puede diagnosticar problemas en las siguientes categorías:

Categoría Descripción Ejemplos
EncendidoProblemas relacionados con el encendido del equipoNo enciende, botón defectuoso
PantallaProblemas de visualización en el monitorPantalla negra, pantalla azul
HardwareFallas en componentes físicosPitidos, ruidos extraños, sobrecalentamiento
SoftwareProblemas de programas y sistema operativoLentitud, virus, controladores
RedProblemas de conectividadSin Internet, WiFi desconectado
RendimientoProblemas de velocidad y respuestaEquipo lento, congelamientos

Reglas de Diagnóstico

El sistema cuenta con {{ total_reglas }} reglas de producción que relacionan síntomas con fallas específicas. Cada regla sigue el formato:

SI (condiciones) ENTONCES (conclusión)

Puede consultar la lista completa de reglas en la sección Reglas del sistema.

Preguntas del Sistema

El sistema cuenta con {{ total_preguntas }} preguntas organizadas en las siguientes categorías:

  • Encendido: ¿El equipo enciende? ¿El botón funciona correctamente?
  • Pantalla: ¿Qué tipo de pantalla muestra? ¿Hay código de error?
  • Sonidos: ¿Se escuchan pitidos o ruidos extraños?
  • Temperatura: ¿El equipo se calienta demasiado?
  • Comportamiento: ¿Se apaga solo? ¿Se congela?
  • Software: ¿Hay instalaciones recientes? ¿Aparecen ventanas emergentes?
  • Red: ¿Tiene conexión a Internet? ¿El WiFi funciona?

Interpretación de Resultados

Al finalizar el diagnóstico, el sistema mostrará:

  • Falla detectada: Nombre y descripción del problema identificado.
  • Gravedad: Nivel de criticidad (Crítico, Medio, Bajo).
  • Confianza: Porcentaje de certeza del diagnóstico.
  • Recomendaciones: Pasos detallados para solucionar el problema.
  • Reglas activadas: Lista de reglas que coincidieron con los síntomas.
Importante: Si la confianza del diagnóstico es baja (menos del 50%), se recomienda consultar con un técnico especializado.

Consejos Útiles

Mantenimiento preventivo

Realice limpieza periódica del equipo para evitar sobrecalentamiento y fallas prematuras.

Actualizaciones

Mantenga el sistema operativo y los programas actualizados para evitar vulnerabilidades.

Copias de seguridad

Realice respaldos periódicos de su información importante para evitar pérdida de datos.

Antivirus

Use un antivirus actualizado y evite descargar archivos de fuentes no confiables.
